# Installation

## Requirements

- Python 3.9 or higher
- pip package manager

## Install from PyPI

```bash
pip install openagent
```

## Install from Source

```bash
git clone https://github.com/GeneticxCln/OpenAgent.git
cd OpenAgent
pip install -e .
```

## Development Installation

```bash
git clone https://github.com/GeneticxCln/OpenAgent.git
cd OpenAgent
pip install -e ".[dev]"
```

## Verify Installation

```bash
openagent --help
```
